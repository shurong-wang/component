## 关键点

#### 自定义事件

1. 本质： 观察者模式
2. 优点： 跳出原生事件的限制，提高封装的抽象性

#### JS 类库分两种

1. utility.js: 与 UI 无关，如 ajax、cookie、drag 等
2. widget.js: 与 UI 有关，如 calendar、rich text box、animation 等

#### 为 Widget 的添加统一的生命周期

1. 团队多人开发的组件风格统一
2. 组件风格统一方便用户使用
